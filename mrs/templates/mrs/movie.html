Content Based Movie Recommendation Engine


Objective: End-to-End lifecycle of a Data Science Project

Tools: Django, sklearn, Pandas, Jupyter Notebook, API

Web Technologies: HTML, CSS, JS, Bootstrap4

Datasets: IMDB 80k and TMDB 5000 Datasets (kaggle)


Implementation:-
Build a Tf-Idf Vectorizer for almost 13,000 english movies based on categorical features of Actor name, Director name and Genres. Used the concept of angular distance for finding the similarity score among movies by calculating cosine similarity.
